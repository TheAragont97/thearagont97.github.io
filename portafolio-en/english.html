HTML 5

Its acronym comes from the English 'HyperText Markup Language'. This markup language allows us to create the basic structure (or body) of a website using tags (labels). The number five (5) refers to the version you are currently on. We have all the titles, paragraphs, forms, videos, among many other things, thanks to this wonderful language.

+

CSS 3

CSS is a style language. Its initials also have a meaning 'Cascading StyleSheet', its translation is 'Cascading Style Sheets'. The number three (3) refers to the current version of said language. It also allows us to style the body of our websites. It is considered as the makeup of HTML. All the design of a site is directly related to this language.

+

JavaScript

JavaScript is a programming language that allows us to add, remove, apply animations or effects to the components of our site. It is the perfect tool to make dynamic and very attractive websites linked to CSS and HTML. Over time it has evolved exponentially and right now it's absolutely necessary to know if you want to be a web developer.

+

Bootstrap

It's one of the most used frameworks in the world. This framework makes the design and positioning of elements with CSS much simpler. You save a lot of time thanks to the fact that by default he brings the tools or internal code so that you do not have to write anything other than specific classes that you have predefined. With one of these bootstrap classes, you could save five (5) or more lines of code in CSS.

Android

It is one of the most widely used languages for developing mobile applications. Android, along with Java or Kotlin, allows you to create native apps that run quickly and smoothly on almost any Android device. It has its own framework, Android Studio, which makes programming, design and testing much easier. In addition, it comes with many ready-to-use tools, such as libraries to connect to the internet, save data or design screens with just a few lines of code. Thanks to its structure and all that offers by default, you can develop a complete app without having to program everything from scratch. Translated with DeepL.com (free version)

Java

It is one of the most popular and widely used programming languages in the world. Java is very versatile, which means that you can use it to create anything from desktop applications to large, complex systems or mobile apps. One of its great advantages is that works on almost any operating system without the need to change the code, thanks to its famous motto: ‘write it once, run it anywhere’. It also has lots of ready-to-use tools and libraries, which saves you time at when programming. Python

It is one of the easiest programming languages to learn and also one of the most powerful. Python is used for almost everything: from automating simple tasks to creating web pages, artificial intelligence or data analysis. Its syntax is very clear and similar to human language, which makes writing code faster and understandable. In addition, it has a huge community and thousands of ready-to-use libraries, so you often don't need to create things from scratch. With a few lines of code in Python you can do what would take much longer in other languages.
